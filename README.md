Simple desktop app to download GitHub release assets without confusion. Automatically selects the best file for your system with a clean, beginner-friendly interface.
# 🚀 EzGD - Easy Git Downloader

**Download the right file from GitHub releases without guessing.**
**Descargá el archivo correcto de GitHub sin tener que pensar.**

---

## 🧠 What is EzGD?

**EzGD (Easy Git Downloader)** is a simple desktop app designed for normal users who just want to download files from GitHub releases.

No technical knowledge required.
No confusion.
No digging through files.

👉 Paste a GitHub link → get the best download → done.

---

## 🧠 ¿Qué es EzGD?

**EzGD (Easy Git Downloader)** es una aplicación de escritorio pensada para usuarios comunes que solo quieren descargar archivos desde GitHub.

Sin conocimientos técnicos.
Sin confusión.
Sin perder tiempo.

👉 Pegás un link → elegís descarga → listo.

---

## ✨ Features / Características

* 🔍 Detects GitHub repositories from multiple URL formats
* 📦 Shows latest release + previous versions
* ⭐ Automatically selects the **recommended file**
* ⬇️ One-click **Download Recommended** button
* 📊 Progress bar with real-time download status
* ❌ Cancel download anytime
* 🧠 Smart detection (avoids ARM, debug, etc.)
* 🧾 Download history (local)
* 🌙 Modern dark UI (customtkinter)
* 📦 Fully portable (.exe with PyInstaller)
